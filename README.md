# Diary

A tiny web-application for collecting daily notes. Build upon jQuery 3, Bootstrap 5, Node.js and PostgreSQL.

## Installation

### Via Docker Compose

The repository provides an example of Docker Compose environment.

```
# cd diary/
# docker compose up -d
```
